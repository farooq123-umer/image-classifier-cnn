# Image Classifier using CNN

A deep learning project that classifies images into 10 categories using the CIFAR-10 dataset.

## What I used
- Python
- TensorFlow, Keras
- NumPy, Matplotlib

## What I did
- Built and trained a CNN from scratch on 50,000 training images
- Used data augmentation and dropout to reduce overfitting
- Achieved around 82% test accuracy
- Experimented with transfer learning using VGG16 which improved accuracy by 9%

## What I learned
This was my first deep learning project. I learned how convolutional layers work and why data augmentation helps the model generalize better.
